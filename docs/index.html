Donots

Send an email for every macOS notification.

Download from GitHub

Requires macOS 14 Sonoma or later.

Features

  • Monitors Notification Center for new notifications
  • Captures a screenshot of the notification region
  • Emails the screenshot via Apple Mail automatically
  • Crosshair tool for selecting the capture region visually
  • JPEG/PNG compression with adjustable quality and scale
  • Menu bar icon with quick start/stop control
  • Separate Activity Log window for monitoring events

Setup

  1. Open Donots and go to Settings > Permissions
  2. Grant Accessibility, Screen Recording, and Mail Automation permissions
  3. Configure your recipient email in the main window
  4. Use the crosshair tool to select your notification capture area
  5. Click Start and you're done
